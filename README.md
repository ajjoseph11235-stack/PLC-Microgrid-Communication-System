# PLC-Based Microgrid Communication and Distributed Load Control System

## Overview
This project implements a PLC-based communication system for microgrid monitoring and load control using Arduino.

## Features
- Voltage sensing and data acquisition
- Structured data packet transmission
- Bidirectional communication
- Relay-based load control
- Scalable microgrid architecture

## Hardware Used
- Arduino Boards
- ACS712 Current Sensor
- ZMPT101B Voltage Sensor
- PLC Modem (Conceptual/Prototype)
- 16x2 LCD
- Relay Module

## System Architecture
Sensor → Microcontroller → PLC Channel → Receiver → Control Unit
